# ✈️ Flight Price Analysis & Model Comparison

This project focuses on analyzing flight price data and comparing multiple Machine Learning models to identify which model performs best.

## 📌 Problem Statement
Flight prices depend on various factors like airline, duration, stops, and journey details.  
The goal of this project is to analyze the data and compare different regression models to determine the best-performing model.

## 📊 Dataset
- Flight price dataset used for analysis
- Features include:
  - Airline
  - Source & Destination
  - Duration
  - Total Stops
  - Date of Journey

## ⚙️ Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Comparison

## 🤖 Models Used
The following regression models were used:

- Linear Regression  
- Lasso Regression  
- Ridge Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- Gradient Boosting Regressor  
- XGBoost Regressor  
- LightGBM Regressor  

## 📈 Objective
The main objective is to compare multiple machine learning models and identify which model performs better on the given dataset.

## 📊 Result
- Compared performance of multiple models  
- Identified the best-performing model based on evaluation metrics
